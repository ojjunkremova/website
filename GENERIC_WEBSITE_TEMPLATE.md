## Image Rules

- Use real job photos whenever possible.
- Prefer clear before and after pairs.
- Remove text-stamped or overlay-heavy images if cleaner originals exist.
- Use descriptive alt text that explains the scene, not just the file type.
- Keep the image framing consistent across pages.

## Content Rules

- Keep copy direct and local.
- Mention the city or region often.
- Show proof before asking for trust.
- Use simple, high-converting CTAs.
- Avoid generic stock-photo language.

## Layout Rules

- Hero first, proof second, services third.
- Keep sections short and scannable.
- Use a strong visual hierarchy.
- Reuse the same layout pattern for each new company.
- Fill in company-specific text, photos, links, and service names only.
